# DELL LCD Enhance
一款让你戴尔服务器的LCD面板更有用的程序!

本项目使用 **GPL3.0** 协议开源

## 目前支持功能
1. 设备相关
    - CPU占用
    - 内存占用
    - 启动时间
    - 硬盘空间
2. Minecraft服务器相关
    - mc服务器人数
    - mc服务器延时
3. mqtt消息
4. get请求http

## 需要pip安装的库
- psutil
- mcstatus
- paho-mqtt
- requests
